# Instagram Agent Extension

## Overview
The Instagram Agent extension is designed to provide users with enhanced functionalities for managing their Instagram accounts. This tool allows users to automate various tasks, improving their experience and efficiency on the platform.

## Features
- **Auto Unfollow**: Automatically unfollow users who do not follow you back.
- **Auto Like**: Like a specified number of posts from a given list of users.
- **Scheduled Posts**: Schedule your posts to be published at specific times.
- **Analytics**: Gain insights into follower growth, engagement rates, and more.

## Installation
1. Download or clone the repository.
2. Install the required dependencies using the following command:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure your Instagram account settings in the `config.json` file.

## Usage
- Run the extension using the provided command:
   ```bash
   python main.py
   ```
- Follow the on-screen instructions to utilize the features.
